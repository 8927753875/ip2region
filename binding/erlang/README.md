# ip2region xdb erlang 查询客户端


### 编译

```
$ rebar3 compile
```
### 运行
启动erlang节点：
```
$ rebar3 shell

```
在erlang节点中执行以下指令查询Ip地址信息：
```
1> ip2region:search("1.0.8.0").
#{city => <<229,185,191,229,183,158,229,184,130>>,
```
